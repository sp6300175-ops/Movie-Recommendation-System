# 🎬 Movie Recommendation System

Hey! I'm Sneha Patel, and this is one of the projects I built 
during my Python with AI Internship Bootcamp.

This was my first real Python project and honestly it was so 
satisfying to see it actually work! The idea was simple — 
instead of spending 20 minutes deciding what to watch, 
just let the program suggest something 😄

## What does it do?
A menu-driven program where you pick a genre and it randomly 
recommends movies from that genre. You can also add your own 
favourite movies and search for any movie in the list.

## What I learned
- How to use dictionaries and lists together
- Writing and calling functions
- Building a simple loop-based menu
- Using Python's random module

## How to run
Open in Google Colab and just run all cells — 
the program will guide you from there!

Python 🐍
